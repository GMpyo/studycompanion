# 🎮 개인 레이드 시스템 기획서 (솔로 토벌전)

**컨셉 요약**
- **목표:** 일반적인 맵 진행 방식에서 벗어나, 사용자가 스스로의 '공부 시간'을 무기 삼아 내면의 방해 요소(보스)를 물리치는 솔로 레이드 구조.
- **전투 방식:** 누적 공부 시간이 보스에게 가해지는 '피해량(Damage)'으로 환산됨 (예: 1시간 = 60 HP 타격).
- **시너지:** 캐릭터의 속성(열정/평정/끈기/지혜)과 역할에 따라 특정 방해 요소에 추가 피해를 입히거나 버프를 제공함.

## ⚔️ 시너지 및 보너스 규칙 (예시)
* **🔥열정:** '무기력/불안' 계열 보스에게 1.2배의 피해. (시간당 타격 효율 증가)
* **💧평정:** '산만/소음' 계열 보스 타격 시, 방해 스킬(예: 집중 시간 차감) 저항 확률 증가.
* **🍃끈기:** '유혹/지루함' 계열 보스 타격 시, 장시간(2시간 이상) 연속 공부 시 콤보 보너스 피해 추가.
* **💡지혜:** '완벽주의/정체기' 계열 보스 타격 시, 휴식 후 복귀 시 '재집중 버프' 부여.
* **역할군:** 딜러(피해량 보너스), 탱커(보스의 방해/디버프 무시), 힐러(지친 캐릭터 휴식 시간 단축), 서포터(파티 전체 경험치/재화 보너스).

## 💻 시스템 구현 아이디어 (UI/UX 및 기술 구조)

### 1. UX/UI 흐름 방향성 (타이머와 전투의 결합)
**혼합형(Hybrid) 방식 추천:** 집중을 해치지 않으면서도 성취감을 극대화하는 방식.
* **공부 진행 중 (타이머 화면):** 화면 상단에 작은 보스 아이콘과 HP 바만 심플하게 노출. 1분마다 데미지 숫자(-1)가 작게 떠오르며 실시간으로 내가 싸우고 있다는 최소한의 시각적 피드백만 제공.
* **공부 종료 후 (결과 정산 화면):** 타이머 종료 버튼을 누르면 **[토벌 결과창]**으로 전환. 누적된 공부 시간이 하나의 거대한 공격 에너지가 되어 보스를 화려하게 타격하는 애니메이션 연출. (이때 캐릭터 속성 시너지 보너스가 곱해지며 "크리티컬!" 연출 발생)

### 2. 치팅 방지 및 상호작용 (보스 스킬 시스템)
그냥 타이머만 켜두는 행위를 방지하기 위해 보스의 '방해 패턴'을 기획적으로 도입합니다.
* **돌발 QTE (Quick Time Event):** 예를 들어 챕터 1의 보스 타격 중, 무작위로 화면에 *"졸음 요정이 수면 가루를 뿌립니다! 1분 내로 화면을 탭해서 깨어나세요!"* 라는 알림 발생.
* **패널티 적용:** 지정된 시간 내에 화면을 탭하지 않으면 타이머가 일시 정지되거나, 해당 세션의 최종 데미지가 반감됨. (하드코어/집중 모드를 켠 유저에게만 적용하고 클리어 보상을 높이는 방식으로 활용)

### 3. 데이터 및 백엔드 처리 흐름
* **HP의 기준:** 1분 = 1 HP로 산정하여 직관성 확보. (보스 체력이 180이라면 3시간 공부 필요)
* **데미지 계산 (서버사이드 로직):**
  1. 클라이언트(앱)에서 타이머 종료 후 `{실제공부시간: 120분, 장착캐릭터_속성: 불}` 데이터를 전송.
  2. 서버(Supabase 등)에서 `기본 데미지(120) * 속성 시너지(1.2배) = 144`를 계산하여 DB의 `current_boss_hp`에서 차감.
  3. 보스 HP가 0 이하가 될 경우, 스테이트를 다음 보스로 자동 업데이트하고 보상 지급 처리.
* **오프라인 보정:** 인터넷 연결이 끊긴 상태에서 공부한 기록은 기기 내부(SQLite/AsyncStorage)에 캐싱해두었다가, 온라인 전환 시 "누적된 데미지가 폭발합니다!" 형태의 일괄 타격 연출로 보완.

### 4. 시각적 연출 및 프론트엔드 요소
* **타격감 강화:** Lottie나 Reanimated를 활용해 피격 시 보스 이미지의 화면 흔들림(Shake), 화면 번쩍임(Flash) 효과 적용.
* **체력바 트랜지션:** 정산 시 보스의 체력이 깎일 때 한 번에 사라지지 않고 쭈우욱 줄어드는(Lerp) 부드러운 애니메이션을 넣어 쾌감 극대화.
